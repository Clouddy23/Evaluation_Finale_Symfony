## Création d'un projet Symfony 7

```bash 
symfony new evaluation-finale-symfony --version=-7.4 -webapp

php bin/console --version
```

## Push du projet symfony sur repo GitHub

```bash 
cd evaluation-finale-symfony

git add .

git commit -m "Initialisation projet Symfony 7"

git remote add origin https://github.com/Clouddy23/Evaluation_Finale_Symfony.git

git push -u origin main
```

## Instructions 
Clone du dépôt de démarrage sur l'IDE VS Code.
Développement de l’Application : Utilisez Symfony 7 pour développer l’application selon les exigences fournies. Gardez à l’esprit les meilleures pratiques de développement, la propreté du code, et l’utilisation efficace des fonctionnalités de Symfony.
Commit et Push sur GitHub : Tout au long de l’examen, vous êtes encouragé à faire des commits réguliers sur votre dépôt GitHub pour sauvegarder votre progression. À la fin de l’examen, assurez-vous de faire un dernier push de tous vos changements sur le dépôt GitHub.
